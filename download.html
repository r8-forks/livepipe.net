Submitting Issues & Patches

LivePipe UI uses GitHub as its issue tracker. You have to sign up first to open a new issue. Please keep the title concise and use appropriate text formatting for the body of the issue. Please link to gists for patches, or check your changes into your clone on GitHub and create a pull request. The issue system is for bug reports and feature requests only.

Discussion Groups

  • LivePipe UI Core - Patches, releases and development of LivePipe UI. Do not post support questions.
  • LivePipe UI Users - Support, general questions, announcements and general discussion.
